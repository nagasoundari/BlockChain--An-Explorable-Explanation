The Block - 'Chain'
As noted previously, the hash of a block renders it immutable and secure. As you can see the hash of each of the block is connected to the previous block. The first block is a bit special because it is not connected to any previous block. This block is called the Genesis Block and the “Previous Hash” value of this block is “0000”. If you tamper with any block, this will cause the data inside the block to change. This causes the hash value of the block to change. This, in turn, will make the subsequent blocks invalid because they no longer store valid hash of the previous block. Thus changing a single block will render the following blocks invalid. Click on any block to tamper with it.
