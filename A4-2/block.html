The Block
Let’s take a closer look at a block. Each block contains some data, the hash of the block and the hash of the previous block. Click on a side of the block to see what information it contains.
The data that is stored inside a block depends on the type of blockchain. The blockchain of a bitcoin, for example, stores the detail about the transaction such as the sender, receiver and the amount of bitcoins.
The hash of the block is its ‘fingerprint’. It identifies a block and all of its contents and is always unique, just like a fingerprint. The hash of the block is calculated using sophisticated algorithms. Changing the data of the block will cause the hash to change. Thus, hashes are useful when you want to detect changes in a block.
The third element inside each block is the hash of the previous block.
